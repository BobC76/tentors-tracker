# Security Policy

## Scope

Ten Tors Tracker is a read-only hobby project — it fetches publicly available results data from tentors.org.uk and displays it. It holds no user accounts, no personal data, and no credentials.

That said, if you find something that could affect visitors (e.g. XSS, data injection from the upstream results pages, or anything in the Vercel API routes), please let me know privately.

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report privately via GitHub's built-in mechanism:
1. Go to the [Security tab](https://github.com/BobC76/tentors-tracker/security)
2. Click **"Report a vulnerability"**

I'll acknowledge within a few days and aim to resolve genuine issues before disclosing publicly.

## Supported versions

Only the current deployment at https://tentors-tracker.vercel.app is maintained. There are no versioned releases with ongoing security support.
